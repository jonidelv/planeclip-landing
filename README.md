# PlaneClip Landing Page

Marketing website for [PlaneClip](https://planeclip.com) — a flight tracking app for iOS.

## Setup

Static site hosted on GitHub Pages with a custom domain (`planeclip.com`).

Just open `index.html` locally or push to GitHub Pages.

## Structure

- `index.html` — Main landing page
- `css/` — Styles
- `js/` — Scripts (App Store link config)
- `assets/` — Images and screenshots
- `CNAME` — Custom domain for GitHub Pages
